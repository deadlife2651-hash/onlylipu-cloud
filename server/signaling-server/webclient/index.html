Cloud Android on Windows

The smoothest Cloud Android experience on a laptop is scrcpy — a real-time, low-latency Android window. One-time setup (2 minutes):

1. Download scrcpy for Windows: github.com/Genymobile/scrcpy/releases
2. Extract, open a terminal in that folder
3. Connect through a secure tunnel:
  ssh -L 5555:localhost:5555 root@SERVER_IP
4. Then run: scrcpy --video-codec=h265 --max-fps=60

Cloud Android also runs full-screen inside the OnlyLipu phone app.
