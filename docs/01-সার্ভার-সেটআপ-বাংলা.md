# ধাপ ১: Vultr সার্ভার সেটআপ (সময়: ~২০ মিনিট)

আপনার সার্ভার: **Tokyo • 149.28.20.169 • 8 vCPU • 32 GB RAM • Ubuntu 26.04**

> ⚠️ আগে একটা জরুরি কাজ: আপনার রুট পাসওয়ার্ড স্ক্রিনশটে ফাঁস হয়েছে।
> সেটআপ শেষে **অবশ্যই** পাসওয়ার্ড বদলাবেন (নিচে ধাপ ৬)।

## ১. Vultr ওয়েব কনসোল খুলুন
1. ফোন বা ল্যাপটপে console.vultr.com → আপনার সার্ভারে ক্লিক।
2. ডান-উপরে **"View Console"** আইকনে ক্লিক — ব্রাউজারেই টার্মিনাল খুলবে।
3. লগইন চাইলে: user `root`, পাসওয়ার্ড আপনার Vultr পেজেরটা।

## ২. এক কমান্ডে সব ইনস্টল
কনসোলে এই কমান্ডটা পেস্ট করে Enter চাপুন:

```bash
curl -fsSL https://raw.githubusercontent.com/deadlife2651-hash/onlylipu-cloud/main/server/install.sh | bash
```

এটা নিজে নিজেই করবে:
- Docker ইনস্টল
- Firewall (শুধু 22/80/443 খোলা)
- Cloud Android (redroid) + Cloud Computer (desktop) কন্টেইনার
- Admin panel + API + HTTPS সার্টিফিকেট (স্বয়ংক্রিয়)

শেষে **✅ INSTALL COMPLETE** দেখলেই সফল।

## ৩. পাসওয়ার্ড দেখুন
```bash
cat /root/onlylipu-credentials.txt
```
এখানে Admin panel-এর ইউজারনেম/পাসওয়ার্ড আর ডেস্কটপের VNC পাসওয়ার্ড আছে।
**স্ক্রিনশট তুলে নিরাপদ জায়গায় রাখুন।**

## ৫. পরে নিজের সাবডোমেইন বসানো
আপনার ডোমেইন প্রোভাইডারে (যেমন GoDaddy) একটা **A record** বানান:
- Host: `cloud` → Value: `149.28.20.169`

তারপর Vultr কনসোলে আবার শুধু এটা চালান:

```bash
DOMAIN=cloud.yourdomain.com bash /opt/onlylipu/install.sh
```

নতুন সার্টিফিকেট নিজে নিজেই বসে যাবে। অ্যাপের সার্ভার URL-ও নতুনটা দিয়ে আপডেট করতে হবে (docs/02 দেখুন)।

## ৬. রুট পাসওয়ার্ড বদলান (জরুরি!)

নতুন শক্তিশালী পাসওয়ার্ড দুইবার দিন।

## সমস্যা হলে

**"binder MISSING" দেখালে (Cloud Android চালু হচ্ছে না):**
```bash
apt-get install -y linux-modules-extra-$(uname -r)
modprobe binder_linux devices="binder,hwbinder,vndbinder"
cd /opt/onlylipu && docker compose up -d
```
তাও না হলে: Vultr → Settings → Change OS → **Ubuntu 24.04 LTS** দিয়ে
রি-ইনস্টল করে install.sh আবার চালান (24.04-এ binder নিশ্চিতভাবে কাজ করে)।

**লগ দেখতে:**
```bash
cd /opt/onlylipu && docker compose logs -f control
```

**সব রিস্টার্ট:**
```bash
cd /opt/onlylipu && docker compose restart
```
