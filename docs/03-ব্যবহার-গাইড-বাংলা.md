# OnlyLipu Cloud — ব্যবহার গাইড (বাংলা)

## প্রথমবার
1. অ্যাপ খুলুন → স্প্ল্যাশ স্ক্রিন → সাইন-ইন।
2. চাইলে ফিঙ্গারপ্রিন্ট আনলক চালু করুন (Settings → Biometric unlock)।

## Dashboard
- **Server status:** Online (সবুজ) / Offline (লাল), পিং (ms), লোকেশন।
- **Cloud Computer কার্ড:** ট্যাপ করলে ডেস্কটপ খুলবে।
- **Cloud Android কার্ড:** ট্যাপ করলে ক্লাউড ফোন স্ক্রিন খুলবে।
- CPU / RAM / Storage বার সার্ভারের অবস্থা দেখায়।
- **Reconnect কার্ড:** শেষ যে পরিবেশ ব্যবহার করেছেন, এক ট্যাপে ফেরত।

## Cloud Android ব্যবহার
- ট্যাপ = টাচ, চেপে ধরা = লং-প্রেস, দুই আঙুল = স্ক্রল।
- নিচের ভাসমান বার: Back / মাউস-টাচ / কিবোর্ড / কোয়ালিটি / সাউন্ড / স্ট্যাটস।
- Back-Home-Recent: ভাসমান বারের **কিবোর্ড** আইকন → নিচে Back/Home/Recent।
- **অ্যাপ ইনস্টল:** Dashboard-এ 🤖 আইকন → **Install App (APK)** →
  ফোন থেকে APK সিলেক্ট → আপলোড % দেখুন → সফল হলে Cloud Android-এর
  অ্যাপ ড্রয়ারে চলে আসবে। Uninstall / Clear data একই পেজে।

## Cloud Computer ব্যবহার
- এক ট্যাপ = বাম-ক্লিক, চেপে ধরা = ডান-ক্লিক, দুই আঙুল = স্ক্রল।
- মাউস/টাচ বাটন দিয়ে ইনপুট মোড বদলান।
- ল্যাপটপে: ব্রাউজারে `/app` খুলে কার্ডে ক্লিক — ফুল-স্ক্রিন ডেস্কটপ।

## কোয়ালিটি সেটিং
সেশনের ভাসমান বারে **HD** আইকন:
Auto / Data Saver / 720p 60 / 1080p 60 / Highest।
দুর্বল নেটে Auto বা Data Saver রাখুন।

## নিরাপত্তা টিপস
- কাউকে পাসওয়ার্ড দেবেন না।
- Settings → **Log out from all devices** মাঝে মাঝে চাপুন।
- Admin panel-এর Audit log-এ সব লগইন/ইনস্টল দেখা যায়।

## সমস্যা
| লক্ষণ | সমাধান |
|---|---|
| "Server offline" | Admin panel → Cloud Android Start; সার্ভার রিবুট: Vultr → Restart |
| স্ক্রিন আসছে না | কোয়ালিটি Data Saver করুন; Wi-Fi বদলান |
| APK ইনস্টল ফেল | 500MB-এর ছোট APK হতে হবে; Admin panel → apps দেখুন VM চালু আছে কিনা |
| লগইন হয় না | পাসওয়ার্ড মিলিয়ে দেখুন credentials ফাইলে |
